# chengyu [![Build Status](https://secure.travis-ci.org/lyuehh/chengyu.png?branch=master)](http://travis-ci.org/lyuehh/chengyu)

chengyu in your command line..

## Getting Started
Install: `npm install -g chengyu`

## Examples

```
$ chengyu -w 游xxx
```

```
游云惊龙
游刃有余
游戏三昧
游戏翰墨
游手好闲
游蜂浪蝶
游骑无归
```

```
$ chengyu  -i 一言
```

```
一言一行
一言为定
一言九鼎
一言半语
一言难尽
季路一言
```
